Debian 13 .bashrc used as a based, stripped from comments.

Added:
- Custom colored prompt,
- `ll` alias (verbose ls),
- `wtd` alias (directory content watcher, cool with tmux),
- `dps` alias (better docker ps, with a touch of color),
- `dockernuke` alias (delete all the docker images, volumes, network, etc.)
